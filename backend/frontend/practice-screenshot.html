Java Selenium 
// Take full page screenshot
TakesScreenshot ts = (TakesScreenshot) driver;
File screenshot = ts.getScreenshotAs(OutputType.FILE);
FileHandler.copy(screenshot, new File("screenshot.png"));

// Take element screenshot
WebElement area = driver.findElement(By.id("screenshotArea"));
File elemShot = area.getScreenshotAs(OutputType.FILE);
FileHandler.copy(elemShot, new File("element_screenshot.png"));

// Screenshot as Base64 (for Extent Reports)
String base64 = ts.getScreenshotAs(OutputType.BASE64);
⚠️ Important
Ensure the element is scrolled into view before Screenshot using element.
