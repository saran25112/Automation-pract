iFrame Practice

Practice switchTo().frame(), interact inside the frame, and switch back to default content

Switch Into The Practice Frame

Interact with the input and button inside the iframe, then read the success message from inside the frame.

Switch into practiceFrame or locate it by id="practiceFrame" before interacting with the elements inside it.
View Selenium Code
Java Selenium 
// Switch into the iframe
driver.switchTo().frame(driver.findElement(By.id("practiceFrame")));

// Interact with elements inside the frame
driver.findElement(By.id("frameNameInput")).sendKeys("Name");
driver.findElement(By.id("frameSubmitBtn")).click();

// Read text inside the frame
String frameMessage = driver.findElement(By.id("frameResult")).getText();

// Return to the main page
driver.switchTo().defaultContent();
⚠️ Important
Ensure the element is scrolled into view before clicking “Submit” within the iframe; otherwise, an ElementClickInterceptedException may occur.
