Key Points

  • Checkbox allows multiple selections
  • Use `click()` to select or unselect
  • Use `isSelected()` to verify state
  • Always check state before clicking as a best practice
  • Use `isEnabled()` to verify whether a checkbox can be interacted with

Java Selenium 
// 1. Basic checkbox
WebElement checkbox = driver.findElement(By.id("subscribe"));
checkbox.click();

// 2. Check if checkbox is selected
boolean isChecked = checkbox.isSelected();
System.out.println(isChecked);

// 3. Select checkbox only if not selected
if (!checkbox.isSelected()) {
    checkbox.click();
}

// 4. Uncheck checkbox
if (checkbox.isSelected()) {
    checkbox.click();
}

// 5. Multiple checkboxes loop example
List<WebElement> checkboxes = driver.findElements(By.name("skills"));
for (WebElement cb : checkboxes) {
    if (cb.getAttribute("value").equals("selenium")) {
        cb.click();
    }
}

// 6. Verify checkbox is enabled
boolean isEnabled = checkbox.isEnabled();
