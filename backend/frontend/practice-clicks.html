Click Targets

Different types of click actions you can perform with Selenium.

View Selenium Code
Java Selenium 
// Single click
driver.findElement(By.id("singleClickBtn")).click();

// Double click using Actions
Actions actions = new Actions(driver);
WebElement dblBtn = driver.findElement(By.id("doubleClickBtn"));
actions.doubleClick(dblBtn).perform();

// Right click (context click)
WebElement rightBtn = driver.findElement(By.id("rightClickBtn"));
actions.contextClick(rightBtn).perform();

// Hover over element
WebElement hoverBtn = driver.findElement(By.id("hoverBtn"));
actions.moveToElement(hoverBtn).perform();
⚠️ Important
Direct click on singleClickBtn may cause ElementClickInterceptedException.
