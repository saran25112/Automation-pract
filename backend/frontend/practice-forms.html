Registration Form

Fill out this form using Selenium text fields, dropdowns, radio buttons, checkboxes, and submit.

View Selenium Code
Java Selenium 
// Fill out the registration form
driver.findElement(By.id("firstName")).sendKeys("John");
driver.findElement(By.id("lastName")).sendKeys("Doe");
driver.findElement(By.id("email")).sendKeys("john@example.com");
driver.findElement(By.id("phone")).sendKeys("+91 9876543210");

// Select radio button
driver.findElement(By.id("genderMale")).click();

// Select from dropdown
Select country = new Select(driver.findElement(By.id("country")));
country.selectByVisibleText("India");

// Check checkboxes
driver.findElement(By.id("hobbyCoding")).click();
driver.findElement(By.id("hobbyReading")).click();

// Enter text in textarea
driver.findElement(By.id("registrationMessage")).sendKeys("Hello!");

// Submit form
driver.findElement(By.id("submitBtn")).click();

Dummy Login Practice

This is a separate practice widget for Selenium only. It does not connect to the real login page.

Practice Only
Use these demo credentials Username: admin Password: Admin@123

Selenium Practice Goal

  • Locate the username and password fields by id.
  • Use sendKeys() to enter the dummy credentials.
  • Click the practice login button and validate the result text.
  • This lab stays isolated from the real login page.
View Dummy Login Selenium Code
Java Selenium 
// Dummy login practice only - not the real login page
driver.findElement(By.id("dummyUsername")).sendKeys("admin");
driver.findElement(By.id("dummyPassword")).sendKeys("Admin@123");
driver.findElement(By.id("dummyLoginSubmitBtn")).click();

// Verify the response text
String loginResult = driver.findElement(By.id("dummyLoginResult")).getText();
Assert.assertTrue(loginResult.contains("Login successful"));
